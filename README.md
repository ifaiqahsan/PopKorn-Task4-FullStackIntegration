# PopKorn: Movie Ticket Booking Platform

PopKorn is a fast, easy, and hassle-free movie ticket booking platform. It features an intuitive interface for browsing movies, selecting seats, and managing bookings[cite: 1].

## Features
* **Movie Browsing**: Explore available movies via a responsive card-based interface[cite: 1].
* **Seat Selection**: Interactive seat maps to streamline the booking process[cite: 1].
* **Seamless Navigation**: Integrated routing for a smooth user experience[cite: 1].

## Tech Stack
* **Frontend**: React, JavaScript[cite: 1].
* **Architecture**: MERN stack (MongoDB, Express, React, Node.js)[cite: 1].
